## Luminir란?

**Luminir**는 HWP, HWPX, PDF, 이미지 문서를 업로드하면 텍스트·표·이미지 정보를 자동으로 추출하고, 추출된 내용을 기반으로 검색 및 AI 질의응답(RAG)을 제공하는 문서 분석 시스템입니다.

한국형 문서(HWP/HWPX)를 AI가 이해할 수 있는 구조화된 데이터로 변환하는 것을 핵심 목표로 합니다.

---

## 주요 기능

| 기능 | 설명 |
| --- | --- |
| 문서 업로드 | HWP, HWPX, PDF, 이미지 파일 업로드 |
| 문서 파싱 | 텍스트, 표, 이미지 정보 자동 추출 |
| 결과 미리보기 | 변환된 문서 내용을 화면에서 바로 확인 |
| 작업 상태 확인 | 문서 변환 진행 상태 실시간 모니터링 |
| 결과 다운로드 | 변환 결과를 파일로 다운로드 |
| RAG 질의응답 | 업로드된 문서 기반 AI 질문·답변 |
| 비동기 처리 | Redis, RabbitMQ 기반 대용량 문서 처리 |
